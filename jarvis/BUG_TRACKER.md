# Bug Tracker

All known bugs, crashes, and UX issues. Populate from Crashlytics, tester feedback,
and direct observation. Triage before fixing — not all bugs block launch.

---

## Priority Definitions

- **P0** — Crash or data loss. Blocks launch. Fix immediately.
- **P1** — Major UX breakage or wrong behavior. Blocks launch if not fixed.
- **P2** — Minor UX issue or edge case. Should fix before launch.
- **P3** — Polish / nice to have. Post-launch acceptable.
